Spotify Clone

A responsive front-end clone of the Spotify Web Player built using HTML and CSS. This project recreates the core layout and design of Spotify, including the sidebar navigation, music library, album cards, and playback controls.

Features
Spotify-inspired user interface
Sidebar navigation with Home, Search, and Library sections
Playlist and album cards with hover effects
Music player control bar at the bottom
Fully responsive design for different screen sizes
Pixel-perfect layout using pure HTML and CSS
Tech Stack
HTML5
CSS3
Font Awesome
Google Fonts
