# Spotify Clone

## Overview

Spotify Clone is a responsive music streaming web application that replicates the core functionality and user interface of Spotify. The application allows users to browse playlists, play and pause songs, navigate between tracks, and control audio playback through an interactive music player.

The project focuses on frontend development and demonstrates the implementation of dynamic user interfaces, audio handling, and responsive design to deliver a seamless music streaming experience.

---

## Features

- Browse and display playlists and songs
- Play and pause audio tracks
- Next and previous track controls
- Interactive progress bar
- Song duration display
- Volume control
- Responsive design for desktop and mobile devices
- Modern Spotify-inspired user interface

## How It Works

1. The application loads a predefined list of songs.
2. Song titles and cover images are displayed in the interface.
3. Users can click the play button to start audio playback.
4. The progress bar updates in real time as the song plays.
5. Users can pause, resume, skip to the next track, or return to the previous track.
6. Album artwork and song details update dynamically based on the selected track.

## Quick Start

### Prerequisites

- Any modern web browser
- Visual Studio Code (optional)
- Git (optional)

---

## 1. Clone the Repository

```bash
git clone https://github.com/Shradha2209/Spotify-Clone.git
cd Spotify-Clone
```

---

## 2. Run the Application

Simply open the `index.html` file in your browser.

### Option 1: Directly Open

Double-click `index.html`.

### Option 2: Using VS Code Live Server

1. Open the project in Visual Studio Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.
